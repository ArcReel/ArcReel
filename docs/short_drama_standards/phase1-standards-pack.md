# Phase 1 Short-Drama / Seedance Standards Pack

This document mirrors the importable prompt-rule constants in `lib/prompt_rules/`.
It is a production checklist for local ArcReel short-drama work; it is not a new
provider integration and does not enable market automation or unattended batch
production.

## Seedance visual prompt policy

- Use only video durations exposed by the current model capability set.
- Keep image prompts visible and single-frame grounded: subject, environment,
  lighting, and atmosphere.
- Keep video prompts to one observable continuous action per shot.
- Preserve registered asset names exactly instead of inventing substitutions.
- Prefer fast vertical short-drama entry: conflict/crisis before exposition.

## Storyboard continuity and C/S/P discipline

- C/S/P means Character / Scene / Prop; all references must map to project assets.
- Consecutive shots should preserve pose, object position, gaze, and light continuity.
- Dependent shots need first-frame / last-frame handoff signals.
- Multi-character shots must keep relative positions and carried props stable.
- Missing core cards/references should become QA findings rather than prompt guesses.

## Creative quality checklist

- Start with hook: crisis, reversal, secret, loss of control, or strong emotion.
- Maintain turn density about every 15 seconds where feasible.
- Mark paywall/follow points with visible irreversible choices or reveals.
- End with satisfaction plus cliffhanger.
- Phase 1 compliance and taste checks warn; only deterministic mechanical failures block.
